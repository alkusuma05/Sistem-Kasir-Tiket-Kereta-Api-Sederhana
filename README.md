# Sistem-Kasir-Tiket-Kereta-Api-Sederhana
Program Python ini mengelola sistem tiket kereta api interaktif. Menggunakan nested dictionary untuk otomatisasi harga berdasarkan rute dan kelas. Dilengkapi validasi try except, sistem menyimpan data ke dalam list, lalu menampilkannya dalam tabel laporan yang rapi. Kode ini menerapkan clean code untuk mempermudah untuk pemula.

- dari soal latihan di modul 4 PDTD (pelatihan dasar teknologi digital)
